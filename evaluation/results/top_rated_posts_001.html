How to concatenate text from multiple rows into a single text string in SQL server?top_rated_posts_001

I/O Example

INPUT: input1

SubjectID:IntStudentName:Str
1Mary
1John
1Sam
2Alaina
2Edward

OUTPUT

SubjectID:IntStudentName:Str
1Mary, John, Sam
2Alaina, Edward

Hints

  • (empty)

Our Solution


SELECT
    SubjectID,
    string_agg(StudentName, ', ') 
FROM
    input1 
GROUP BY
    SubjectID 
ORDER BY
    SubjectID ASC
