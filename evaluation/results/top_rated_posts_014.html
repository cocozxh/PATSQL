How to use GROUP BY to concatenate strings?top_rated_posts_014

I/O Example

INPUT: input1

id:Intstring:Str
1A
1B
2C

OUTPUT

id:Intstring:Str
1A, B
2C

Hints

  • (empty)

Our Solution


SELECT
    id,
    string_agg(string, ', ') 
FROM
    input1 
GROUP BY
    id 
ORDER BY
    id ASC
