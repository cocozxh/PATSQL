How to count occurrences of a column value efficiently in SQL?top_rated_posts_019

I/O Example

INPUT: input1

id:Intage:Int
025
125
223

OUTPUT

id:Intage:Intcount:Int
0252
1252
2231

Hints

  • (empty)

Our Solution


SELECT
    T0.id,
    T1.age,
    T1.count_id 
FROM
    input1 AS T0 
JOIN
    (
        SELECT
            age,
            count(id) AS count_id 
        FROM
            input1 
        GROUP BY
            age
    ) AS T1 
        ON T1.age = T0.age 
ORDER BY
    T0.id ASC
