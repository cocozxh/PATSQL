How to concatenate strings of a string field in a PostgreSQL 'group by' query?top_rated_posts_016

I/O Example

INPUT: input1

ID:IntCOMPANY_ID:IntEMPLOYEE:Str
11Anna
21Bill
32Carol
42Dave

OUTPUT

COMPANY_ID:IntEMPLOYEE:Str
1Anna, Bill
2Carol, Dave

Hints

  • (empty)

Our Solution


SELECT
    COMPANY_ID,
    string_agg(EMPLOYEE, ', ') 
FROM
    input1 
GROUP BY
    COMPANY_ID 
ORDER BY
    COMPANY_ID ASC
