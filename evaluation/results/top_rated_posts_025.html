How to get N rows starting from row M from sorted table in T-SQLtop_rated_posts_025

I/O Example

INPUT: input1

Id:StrValue:Str
1a
2b
3c
4d
5e
6f

OUTPUT

Id:StrValue:Str
3c
4d

Hints

  • 3:Int
  • 5:Int

Our Solution


SELECT
    T0.Id,
    T0.Value 
FROM
    (SELECT
        Id,
        Value,
        rank() OVER (ORDER BY Id ASC) AS rank_over_order_by_id_asc 
    FROM
        input1) AS T0 
WHERE
    T0.rank_over_order_by_id_asc >= 3 
    AND T0.rank_over_order_by_id_asc < 5 
ORDER BY
    T0.Id ASC
