# solution

```sql
SELECT EXTRACT(MONTH FROM trip_start_timestamp) AS month, 
       COUNT(1) AS num_trips
FROM `bigquery-public-data.chicago_taxi_trips.taxi_trips`
WHERE EXTRACT(YEAR FROM trip_start_timestamp) = 2017
GROUP BY month
ORDER BY month
```
